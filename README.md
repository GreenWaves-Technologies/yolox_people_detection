# YOLOX detection GAP inference

This repository contains the code for the inference of [YOLOX](https://arxiv.org/pdf/2107.08430.pdf) on GVSOC


# Setup

```bash
cd <GAP_SDK_HOME>
source sourceme.sh 
```

Select option 3 from the menu



# GVSOC inference

```bash
cd inference_gvsoc
make all run platform=gvsoc
```
